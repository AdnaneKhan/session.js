`GroupManager.sendMessage(groupPubKey, text?, { expireTimer? })` (plan §2.3):

- Guards: group must be known (`GroupNotFoundError`) and active

  (`InvalidKeypairError`).

  namespace −10, `GroupContext{id:utf8(addr), DELIVER}` — core covered in
  G2-T3/G2-T4).
- Honors the group's `deleteAfterSend` timer (per-group `expireTimer` override
  or the group's stored `expirationTimer`).
- Receive side: `#handleGroupMessage` now only surfaces chat for groups we are
  an **active** member of (unknown / left groups dropped).
