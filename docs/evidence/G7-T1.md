## What

`GroupManager` reconciles an inbound legacy config sync (the `syncClosedGroups`
event emitted by the core poller from `ConfigurationMessage.closedGroups`,
parsed in G2-T4), spec §2.6:

- **join** groups we don't know (store state, append keypair, start polling,
  emit `groupJoined`);
- **overwrite** the state of groups we do (config is authoritative) and append
  the carried keypair (dedupe-by-value);
- **delete** active groups absent from a **non-empty** sync (a linked device
  left/was removed — no leave message), emitting `groupRemoved`;
- guard: an **empty** config does not wipe known groups.
