## Containment guarantees verified

```
cd calls && bun test test/call-manager.test.ts    # exit 0 — 14 pass (incl. containment block)
cd calls && bun test test/reconnect.test.ts       # chaos block — 0 unhandled rejections
```

1. **Media engine createSession throws** during `call()` → placeCall fails
   the just-registered context → "error" event WITH the Call handle →
   call() rejects with the MediaFailureError; call ended `error`;
   `activeCall` cleared.
2. **Consumer onAudio handler throws** on delivered audio → CallImpl wrapper
   contains → supervisor.failCall → "error" event (attributed to the Call) +
   call ended `error`. The real AudioBridge receive path ALSO contains
   consumer throws (defense in depth — media layer).
3. **session.sendCallMessage rejects** (signaling plane down) → critical-path
   send failure ends the failing call with an error event (supervisor
   #onSignalingError → failCall) instead of limping into the timeout; late
   async rejections racing teardown surface the error but never end a call
   twice (contexts-registry guard).
4. **Process-level assertion**: a `process.on("unhandledRejection")` listener
   attached across all three failure scenarios — count === 0.
5. **Chaos** (reconnect.test.ts): 200 random events (garbage messages,
   self-messages, stale timestamps, mismatched ICE parallel arrays, media
   state flapping incl. "failed", hangup/accept races, timer advances) →
   zero throws, zero unhandled rejections.
6. CallImpl misuse (accept/reject/ignore on outbound) rejects/throws typed
   InvalidCallMessageError without disturbing the call; CallContext.ignore()
   can no longer leak an unhandled rejection (attached catch).

## Typed-error propagation summary

| Failure | Error | EndReason |
|---|---|---|
| call() bad Session ID | InvalidCallMessageError | n/a (rejected promise) |
| call() while active | CallInProgressError | n/a |
| call() unapproved peer | PeerNotApprovedError | n/a |
| createSession throw | MediaFailureError | error |
| signaling critical-path reject | wrapped/`SIGNALING_FAILURE` | error |
| ICE restart exhaustion | IceFailureError | ice-failed |
| media state "failed" | MediaFailureError | ice-failed |
| consumer callback throw | original Error | error |
