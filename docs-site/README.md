## Made for Session.js

Use Session messenger programmatically with [Session.js](https://github.com/AdnaneKhan/session.js): Session bots, custom Session clients, and more.

## This fork

This site is the MIT-licensed upstream
[session.js docs](https://git.hloth.dev/session.js/docs) (© Viktor
Shchelochkov) merged into the fork as a git subtree, deployed to GitHub
Pages by [`.github/workflows/docs.yml`](../.github/workflows/docs.yml) on
every push to `main` that touches `docs-site/`. To pull in upstream doc
updates later:

```sh
git subtree pull --prefix=docs-site docs-upstream main --squash
```
